    Workflow

    1. Comprendre les exigences

    → Lire 01_EXIGENCES_GOOGLE.md

    2. Suivre la campagne

    → Cocher 02_CHECKLIST.md
    → Mettre à jour 03_SUIVI_CAMPAGNE.md (1-2x/semaine)

    3. Préparer la soumission

    → Remplir 04_FORMULAIRE_PRODUCTION.md
    → S'assurer que le tableau "Feedback → Corrections" est complet dans 03_SUIVI_CAMPAGNE.md

    4. Soumettre

    → Copier les réponses de 04_FORMULAIRE_PRODUCTION.md dans le formulaire Google
